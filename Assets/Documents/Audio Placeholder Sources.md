# Audio Placeholder Sources (Steam-Safe Workflow)

You can ship placeholders now if licenses are tracked correctly.

## Recommended Source Order

1. Kenney
- Best default for quick prototype audio packs
- Usually very permissive and easy to track

2. OpenGameArt
- Large library
- Check each asset's exact license before use

3. Freesound
- Useful for niche effects
- Only use files with clear commercial-safe license

## Import Rules

- Keep a project log of every imported audio file:
  - File name
  - Source URL
  - License type
  - Author attribution text (if required)
- Store this in a single licenses/credits document
- Do not use assets with unclear license terms
